---
title: "Requesting Access"
description: "How to get an API key."
---

Developer API access is granted on request rather than self-service signup, so
we can scope access correctly and help size your integration. Once access is
granted, **you generate your own API key** from the Developer Portal — your key
is only ever shown to you, never handled by our team.

<Steps>
  <Step title="Submit the request form">
    Fill out the [access request form](https://developer.titletrackr.com/request)
    with your company, contact details, the products you need (Orders, File
    Vault, or both), and your expected volume.
  </Step>
  <Step title="We review and grant access">
    Our team reviews the request and enables the Developer API for your
    TitleTrackr workspace. We'll let you know when it's ready.
  </Step>
  <Step title="Generate your key">
    Sign in to TitleTrackr and open the **Developer Portal → Tokens** page.
    Click **Create API key**, give it a name, and copy the key — for security it
    is shown only once. Store it somewhere safe.
  </Step>
</Steps>

<Note>
  Already a TitleTrackr customer? Mention your workspace name in the request so
  we can enable access on the right account.
</Note>

You can rotate or revoke your keys, and monitor your usage, anytime from the
**Developer Portal** inside the TitleTrackr app.
