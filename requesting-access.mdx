---
title: "Requesting Access"
description: "How to get an API key."
---

Developer API access is granted on request rather than self-service signup, so
we can scope your key correctly and help size your integration.

<Steps>
  <Step title="Submit the request form">
    Fill out the [access request form](https://developer.titletrackr.com/developers)
    with your company, contact details, the products you need (Orders, File
    Vault, or both), and your expected volume.
  </Step>
  <Step title="We review and provision">
    Our team reviews the request, matches it to your TitleTrackr workspace, and
    issues a scoped API key.
  </Step>
  <Step title="Receive your key">
    We'll share your API key securely. Store it somewhere safe — it is shown
    only once.
  </Step>
</Steps>

<Note>
  Already a TitleTrackr customer? Mention your workspace name in the request so
  we can attach the key to the right tenant.
</Note>

Once provisioned, you can monitor your usage anytime from the **API Usage**
dashboard inside the TitleTrackr app.
